Thuộc tính class trong html
        Thuộc tính class được sử dụng để gom các thẻ html vào 1 nhóm theo ý muôn
        Thẻ html có thể chung class
        Khi mà có nhiều class trong 1 thẻ thì viết cách nhau ra ví dụ
        class="class1 class2 class3 class4"

        Trong CSS muốn chọn được các thẻ dựa trên class
        thì sẽ sử dụng dấu . ví dụ : .mienbac hay .trunguong

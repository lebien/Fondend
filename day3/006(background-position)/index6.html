Thuộc tính background-position ( đặt vị trí cho ảnh nền )
        Làm quen với 4 giá trị vị trí cho thuộc tính này
        top ( bên trên )
        bottom ( bên dưới )
        center ( ở giữa )
        left ( trái )
        right ( phải )
        Đặt vị trí ảnh nền trong 1 thẻ có kích cỡ lớn hơn nó thì sẽ phải
        dựa vào trục X và trục Y
        trục X vị trí xuất hiện theo chiều ngang
        trục Y là vị trí xuất hiện theo chiều dọc

        Chú ý để cho thuộc tính background-position có tác dụng :
        1 - kích cỡ của thẻ html phải lớn hơn hình ảnh nền
        2 - thuộc tính background-repeat phải có giá trị là no-repeat
    
        Các thuộc tính background CSS được sử dụng để xác định các hiệu ứng nền cho các yếu tố.

        tính chất nền CSS:

        màu nền
        hình nền
        Bối cảnh Lặp lại
        background-attachment
        background-position
         1. background-color ( màu nền )
        2. background-image ( ảnh nền )
        3. background-repeat ( có nên lặp ảnh nền khi mà kích thước lớn
        hơn ảnh nền )
        4. background-position ( xác định vị trí của ảnh nền )
        5. background-attachment ( ảnh nền có cố định trong 1 thẻ html hay không )


        Chú ý :
        trong các thuộc tính CSS luôn có giá trị mặc định
        ví dụ thuộc tính background-color ( không màu )
        thuôc tính background-image ( none ) không có ảnh nền
        thuộc tính background-repeat ( repeat ) luôn luôn lặp lại
        Khi giá trị mặc định có viết hay không cũng không có tác dụng
         1. background-color ( màu nền )
        2. background-image ( ảnh nền )
        3. background-repeat ( có nên lặp ảnh nền khi mà kích thước lớn
        hơn ảnh nền )
        4. background-position ( xác định vị trí của ảnh nền )

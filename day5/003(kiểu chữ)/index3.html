Nhúng các font vào thông qua @font-face
        @font-face {
          font-family: "khai báo tên font mà mình mong muốn";
          src: url("đường dẫn của file font");
        }
        ví dụ :
        @font-face {
          font-family: "Lato";
          src: url("Lato/Lato-Regular.ttf");
        }
